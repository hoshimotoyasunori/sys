# ユーザーマニュアル

## 1. はじめに

### 1.1 システム概要
システム設計アシスタントは、プロジェクト管理とタスク管理を効率的に行うためのWebアプリケーションです。

### 1.2 対象ユーザー
- プロジェクトマネージャー
- 開発チームメンバー
- プロジェクト関係者

### 1.3 推奨環境
- **ブラウザ**: Chrome 90以上、Firefox 88以上、Safari 14以上
- **画面解像度**: 1280x720以上推奨
- **インターネット接続**: 安定した接続が必要

## 2. ログイン・ログアウト

### 2.1 初回ログイン

1. **アプリケーションにアクセス**
   - ブラウザでアプリケーションURLにアクセス
   - ログイン画面が表示されます

2. **アカウント作成**
   - 「新規登録」ボタンをクリック
   - メールアドレスとパスワードを入力
   - 「登録」ボタンをクリック

3. **メール認証**
   - 登録したメールアドレスに認証メールが送信されます
   - メール内の認証リンクをクリック

4. **ログイン**
   - 認証完了後、メールアドレスとパスワードでログイン

### 2.2 通常ログイン

1. **ログイン画面で認証情報を入力**
   - メールアドレス
   - パスワード

2. **ログインボタンをクリック**
   - 認証成功時：ダッシュボードに移動
   - 認証失敗時：エラーメッセージが表示

### 2.3 ログアウト

1. **ユーザーメニューを開く**
   - 画面右上のユーザーアイコンをクリック

2. **ログアウトを選択**
   - メニューから「ログアウト」をクリック
   - ログイン画面に戻ります

## 3. プロジェクト管理

### 3.1 プロジェクト一覧の表示

1. **ダッシュボードにアクセス**
   - ログイン後、自動的にダッシュボードが表示されます

2. **プロジェクト一覧の確認**
   - 参加しているプロジェクトが一覧表示されます
   - プロジェクト名、説明、メンバー数が表示されます

### 3.2 新規プロジェクトの作成

1. **新規プロジェクトボタンをクリック**
   - ダッシュボードの「新規プロジェクト」ボタンをクリック

2. **プロジェクト情報を入力**
   - **プロジェクト名**: 必須項目（100文字以内）
   - **説明**: 任意項目（1000文字以内）

3. **プロジェクトを作成**
   - 「作成」ボタンをクリック
   - 作成したプロジェクトが一覧に追加されます

### 3.3 プロジェクトの編集

1. **プロジェクト詳細画面にアクセス**
   - プロジェクト一覧から編集したいプロジェクトをクリック

2. **編集ボタンをクリック**
   - プロジェクト詳細画面の「編集」ボタンをクリック

3. **情報を変更**
   - プロジェクト名や説明を変更

4. **変更を保存**
   - 「保存」ボタンをクリック

### 3.4 プロジェクトの削除

1. **プロジェクト詳細画面にアクセス**
   - 削除したいプロジェクトをクリック

2. **削除ボタンをクリック**
   - 「削除」ボタンをクリック

3. **削除の確認**
   - 確認ダイアログで「削除」をクリック
   - **注意**: 削除したプロジェクトは復元できません

## 4. メンバー管理

### 4.1 メンバーの招待

1. **プロジェクト詳細画面にアクセス**
   - メンバーを招待したいプロジェクトをクリック

2. **メンバー招待ボタンをクリック**
   - 「メンバー招待」ボタンをクリック

3. **招待情報を入力**
   - **メールアドレス**: 招待するユーザーのメールアドレス
   - **権限**: owner（オーナー）、admin（管理者）、member（メンバー）

4. **招待を送信**
   - 「招待」ボタンをクリック
   - 招待メールが送信されます

### 4.2 招待の受け入れ

1. **招待メールを確認**
   - 招待メールの「招待を受け入れる」リンクをクリック

2. **アカウント作成・ログイン**
   - 未登録の場合：アカウントを作成
   - 既登録の場合：ログイン

3. **プロジェクトに参加**
   - 招待を受け入れると、プロジェクトに参加できます

### 4.3 メンバー権限の変更

1. **プロジェクト詳細画面にアクセス**
   - メンバー管理画面を開く

2. **メンバー一覧から対象を選択**
   - 権限を変更したいメンバーをクリック

3. **権限を変更**
   - 権限ドロップダウンから新しい権限を選択
   - 「保存」ボタンをクリック

### 4.4 メンバーの削除

1. **プロジェクト詳細画面にアクセス**
   - メンバー管理画面を開く

2. **削除対象を選択**
   - 削除したいメンバーの「削除」ボタンをクリック

3. **削除を確認**
   - 確認ダイアログで「削除」をクリック

## 5. タスク管理

### 5.1 タスク一覧の表示

1. **プロジェクト詳細画面にアクセス**
   - タスクを管理したいプロジェクトをクリック

2. **タスクタブをクリック**
   - プロジェクト詳細画面の「タスク」タブをクリック

3. **タスクボードの確認**
   - 未着手、進行中、完了の3つの列でタスクが表示されます

### 5.2 新規タスクの作成

1. **新規タスクボタンをクリック**
   - タスクボードの「新規タスク」ボタンをクリック

2. **タスク情報を入力**
   - **タスク名**: 必須項目（200文字以内）
   - **説明**: 任意項目（2000文字以内）
   - **ステータス**: 未着手、進行中、完了
   - **優先度**: 低、中、高
   - **担当者**: プロジェクトメンバーから選択
   - **期限**: 任意項目

3. **タスクを作成**
   - 「作成」ボタンをクリック

### 5.3 タスクの編集

1. **タスクをクリック**
   - 編集したいタスクをクリック

2. **編集モードに切り替え**
   - タスク詳細画面の「編集」ボタンをクリック

3. **情報を変更**
   - タスク名、説明、ステータスなどを変更

4. **変更を保存**
   - 「保存」ボタンをクリック

### 5.4 タスクのステータス変更

#### 方法1: ドラッグ&ドロップ
1. **タスクをドラッグ**
   - タスクカードをマウスでドラッグ

2. **別の列にドロップ**
   - 目的のステータス列にドロップ
   - タスクのステータスが自動的に更新されます

#### 方法2: 編集画面から変更
1. **タスクを編集**
   - タスク詳細画面でステータスを変更

2. **保存**
   - 「保存」ボタンをクリック

### 5.5 タスクの削除

1. **タスク詳細画面にアクセス**
   - 削除したいタスクをクリック

2. **削除ボタンをクリック**
   - 「削除」ボタンをクリック

3. **削除を確認**
   - 確認ダイアログで「削除」をクリック

### 6.2 新規成果物の作成

2. **成果物情報を入力**

   - **説明**: 任意項目
   - **ステータス**: 未着手、進行中、完了
   - **担当者**: プロジェクトメンバーから選択
   - **期限**: 任意項目

3. **成果物を作成**
   - 「作成」ボタンをクリック

### 6.3 成果物の編集・削除

- タスクと同様の手順で編集・削除が可能です

## 7. フェーズ管理

### 7.1 フェーズ一覧の表示

1. **プロジェクト詳細画面にアクセス**
   - フェーズを確認したいプロジェクトをクリック

2. **フェーズタブをクリック**
   - プロジェクト詳細画面の「フェーズ」タブをクリック

3. **フェーズ概要の確認**
   - 各フェーズの進捗状況が表示されます

## 8. リアルタイム機能

### 8.1 リアルタイム更新

- プロジェクトメンバーが行った変更は、リアルタイムで反映されます
- ページの更新は不要です

### 8.2 通知機能

- 新しいタスクが割り当てられた場合、通知が表示されます
- プロジェクトに招待された場合、通知が表示されます

## 9. トラブルシューティング

### 9.1 よくある問題と解決方法

#### ログインできない
- **原因**: メールアドレスまたはパスワードが間違っている
- **解決方法**: 正しい認証情報を入力してください

#### プロジェクトが表示されない
- **原因**: プロジェクトに参加していない、または権限がない
- **解決方法**: プロジェクトオーナーに招待を依頼してください

#### タスクの編集ができない
- **原因**: 権限が不足している
- **解決方法**: プロジェクト管理者に権限の変更を依頼してください

#### リアルタイム更新が動作しない
- **原因**: インターネット接続の問題
- **解決方法**: ページを再読み込みしてください

### 9.2 サポート連絡先

- **技術的な問題**: システム管理者にお問い合わせください
- **使用方法の質問**: プロジェクト管理者にお問い合わせください

## 10. ショートカットキー

| キー | 機能 |
|------|------|
| Ctrl + N | 新規プロジェクト作成 |
| Ctrl + T | 新規タスク作成 |
| Ctrl + K | 検索 |
| Ctrl + S | 保存 |
| Escape | ダイアログを閉じる |

## 11. ベストプラクティス

### 11.1 プロジェクト管理

- **プロジェクト名**: 分かりやすい名前を付ける
- **説明**: プロジェクトの目的を明確に記載
- **メンバー招待**: 適切な権限を設定

### 11.2 タスク管理

- **タスク名**: 具体的で分かりやすい名前を付ける
- **説明**: 詳細な要件や注意事項を記載
- **期限**: 現実的な期限を設定
- **担当者**: 適切な担当者を割り当て

### 11.3 成果物管理

- **成果物名**: 成果物の内容が分かる名前を付ける
- **期限**: プロジェクトスケジュールに合わせて設定
- **ステータス**: 定期的に更新

このマニュアルを参考に、効率的にプロジェクト管理を行ってください。
